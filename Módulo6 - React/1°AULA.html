INTRODUÇÃO

O QUE É REACT

React é uma biblioteca Javascript para criação de interfaces para usuário.

A princípal função do React é organizar o conteúdo que será mostrado ao usuário.

Mesmo podendo ser usado de outras formas(veremos mais adiante), o princípal uso do React em dia é para criar sites, ou interfaces web.

O React é uma das princípais ferramentas do mercado para criação de SPA's. SPA's(Single Page Application) são aplicações web(sites) que tem como objetivo trazer uma experiência similar a de um programa de computador para o usuário(sem os reloads e etc comuns aos sites).

HISTÓRIA DO REACT

Essa biblioteca surgiu em 2011, no facebook, e passou a ser utilizada na interface do mural de notícias da rede social.

No ano seguinte, passou a integrar também a área de tecnologia do instagram e de várias outras ferramentas da empresa.

O react tem sido usado por grandes companhias ao redor do mundo.

Algumas delas: Netflix, Airbnb, American Express, Facebook, Whatsapp, Ebay, Instagram, Globo, Hotmart, Ifood, dentre outras..

COMO FUNCIONA O REACT

Para entender como REACT funciona é preciso conhecer a estrutura de uma página HTML e de que formas ela é manipulada. A página HTML é formada por uma série de tags que representam cada elemento de um conteúdo. Já o DOM(Document Object Model), é uma representação estruturada de todos os elementodo da página e é utilizado para facilitar as alterações nesses elemento

O REACT cria uma Dom virtual para permitir a manipulação dos elementos da página. Basicamente, ele realiza uma cópia do DOM em memória, faz as alterações necessárias conforme a determinação das instruções programadas e atualiza o navegador de forma inteligente. Esse processo oferece mais agilidade na manipulação das páginas.

REACT X REACT NATIVE.

REACT NATIVE e REACTJS são a mesma coisa? Ambos possuem princípios estruturais, maas não idênticos.

O objetivo do REACT NATIVE é permitir a construção de interfaces de aplicativos para celulares, enquanto o foco do REACT.JS são interfaces web (sites).

O REACT NATIVE é uma biblioteca do REACT.JS, adaptada para construir aplicativos para dispositivos móveis, Android e IOS, onde permite os desenvolvedores usem Javascript para criar APP's para celulares.
